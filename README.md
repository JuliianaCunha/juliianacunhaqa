# Contract Signature Testing

Projeto demonstrando minha abordagem para testes de uma funcionalidade de assinatura eletrônica de contratos.

## Objetivo

Garantir que o fluxo de assinatura funcione corretamente em diferentes cenários, assegurando integridade dos dados, regras de negócio e experiência do usuário.

## Atividades realizadas

- Planejamento de testes
- Criação de casos de teste
- Testes funcionais
- Testes exploratórios
- Testes de API
- Validação em banco de dados (SQL)
- Registro e acompanhamento de bugs
- Evidências dos testes

## Tecnologias

- Azure DevOps
- Postman
- SQL Server
