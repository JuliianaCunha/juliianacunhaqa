# Contract Signature Testing

Project demonstrating my approach to testing an electronic contract signature feature.

## Objective

Ensure that the signature workflow works correctly across different scenarios, maintaining data integrity, business rules, and user experience.

## Activities Performed

- Test planning
- Test case design
- Functional testing
- Exploratory testing
- API testing
- Database validation (SQL)
- Bug reporting and tracking
- Test evidence documentation

## Technologies

- Azure DevOps
- Postman
- SQL Server
